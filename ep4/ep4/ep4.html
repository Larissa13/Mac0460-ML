def plot_curve(estimator, data, label, title):
    train_sizes, train_scores, test_scores = learning_curve(estimator, data, label, cv=5, n_jobs=4)
    plt.figure()
    plt.title(title)
    plt.ylim(ymin = 0.7, ymax = 1.1)
    plt.xlabel("Tamanho do conjunto de treinamento")   
    plt.ylabel("Score")
    plt.plot(train_sizes, np.mean(train_scores, axis=1), 'o-', color = 'r', label = "Score de treinamento")
    plt.plot(train_sizes, np.mean(test_scores, axis=1), 'o-', color = 'b', label= 'Score de Teste')
    plt.legend(loc="best")
    return plt

Comentários:¶
Podemos perceber que o C (que  o parmetro de penalidade) do SVM com kernel linear (0.01)  menor que o C do SVM com kernel rbf (5), alm disso, o Eout deste  menor que o do outro, mostrando que quanto maior o valor do C, menor a probabilidade de haver amostras classificadas erroneamente. Atuando, assim, como um fator de regularizao, ajudando a evitar o overfitting. J o gama influencia no quo longe duas amostras podem estar e mesmo assim serem classificadas como do mesmo tipo. Quanto menor o gama, maior esta distncia pode ser. Neste caso, como s temos dois tipos de dados, e eles variam bastante,  razovel que o gama seja um pouco menor. Testei gamas maiores, porm o programa se mostrava muito lento, e mesmo assim, por validao cruzada, o gama escolhido era 0.01. Da decidi por testar os gamas acima.
